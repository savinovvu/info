/ - root
/bin -  основные утилиты для всех пользователей(cat, ls, cp...)
/sbin- Основные системные программы для администрирования и настройки системы, # init, iptables, ifconfig
/boot - загрузочные файлы. файлы загрузчика, ядро, initd,system.map
/dev (devices) - основные файлы устройств (/dev/null, /dev/zero)
/etc - общесистемные конфигурационные файлы
/home содержаит домашние каталоги пользователей, которые в свою очередь содержат персональные настройки и данные пользователя
/lib основные библиотеки, необходимые для работы программ из /bin, /sbin
/media - Точки монтирования сменных носителей, CD_ROM
/mnt - Содержит временно монтируемые файловые системы
/opt (optional) - Дополнительное программное обеспечение
/proc - Виртуальная файловая система, представляющая состояние ядра операционной системы и запущенных процессов в виде файлов
/root - Домашний каталог пользователя root
/run - информация о системе с момента её загрузки, в том числе данные, необходимые для работы демонов (pid-файлы, UNIX- сокеты и тд)
/srv - Данные для сервисов, предоставляемых системой # www, ftp
/sys - Содержит информацию об устройствах, драйверах, а так же некоторых свойств ядра
/tmp - временные файлы
/usr - Вторичная иерархия для данных пользователя. Содержит большинство пользовательских приложений и утилит используемых в многопользовательском режиме
/var - Изменяемые файлы, такие как файлы регистрации, временные почтовые файлы, файлы спулеров
