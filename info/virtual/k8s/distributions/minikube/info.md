
##### Version
```shell script
minikube version
```

#### start
```shell script
minikube start --wait=false
```
#### get ip
```shell script
minikube version
```

#### install ingress
```shell script
minikube addons enable ingress
```

#### install kubernetes dashboard
```shell script
minikube addons enable dashboard
```
#### open dashboard in browser
```shell script
minikube dashboard
```
